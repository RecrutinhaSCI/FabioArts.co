Configurações do Site

Edite o conteúdo das principais seções do site público.

📇 Geral

Marca, contato e redes sociais. Aparece no header e footer.

🎯 Hero

Primeira seção do site — kicker, título grande, subtítulo e 2 botões principais.

👤 Sobre

Texto institucional do "Sobre mim". Para os números/estatísticas, use a página Conteúdo do site → Sobre.

💬 Mentoria

Texto + CTAs da seção Mentoria. Para os bullets/features, use Conteúdo do site → Mentoria.

⚙️ Processo Criativo

Header da seção. Os passos individuais são editados em Conteúdo do site → Processo.

🦶 Footer

Texto de copyright. Para colunas e links, use Conteúdo do site → Footer.
