Configurações do Site

Edite o conteúdo das principais seções do site público.

📇 Geral

Marca, contato e redes sociais. Aparece no header e footer.

🎯 Hero

Primeira seção do site — kicker, título grande, subtítulo e 2 botões principais.

👤 Sobre

Texto institucional do "Sobre mim". Para os números/estatísticas, use a página Conteúdo do site → Sobre.

💬 Mentoria

Texto + CTAs da seção Mentoria. Para os bullets/features, use Conteúdo do site → Mentoria.

⚙️ Processo Criativo

Header da seção. Os passos individuais são editados em Conteúdo do site → Processo.

🖼️ Header — Portfólio

Texto que aparece acima dos cards da seção Portfólio.

🛠️ Header — Serviços

Header da seção Serviços. Os cards são editados em Serviços.

🎓 Header — Cursos

Header da seção Cursos. Os cards são editados em Cursos.

⭐ Header — Avaliações

Header da seção "Avaliações no Google". Para nota/contagem/place ID, use a seção ⭐ Avaliações do Google abaixo.

🎯 CTA Final

Bloco final de conversão. Para os botões, use CTAs (keys cta-primary / cta-secondary).

⭐ Avaliações do Google

Dados do perfil do Google para exibir prova social no site público. Sem chave da API, o site usa os valores manuais abaixo como fallback.

ID do local no Google Places.
Link para ver todas as avaliações no Google.
Link direto para o cliente avaliar no Google.
Nota manual de fallback, exemplo: 5.0.
Quantidade manual de avaliações, exemplo: 61.
🦶 Footer

Texto de copyright. Para colunas e links, use Conteúdo do site → Footer.
